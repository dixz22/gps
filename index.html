🚫 Izin Lokasi Ditolak

Browser memblokir akses lokasi. Untuk mengaktifkannya kembali:
— Chrome/Edge: klik ikon 🔒 di address bar → Izin situs → aktifkan Lokasi
— Firefox: klik ikon 🛡️ → Hapus blokir → muat ulang halaman
— Safari: Pengaturan → Safari → Lokasi → Izinkan

Tentang Koordinat.id

Tool ini mendeteksi koordinat GPS presisi tinggi menggunakan Geolocation API browser. Tidak ada server, tidak ada tracking, dan tidak ada data yang disimpan dari sisimu.

Alamat wilayah dideteksi melalui 6 layer resolusi menggunakan OpenStreetMap (Nominatim) dan BigDataCloud — gratis, tanpa API key.

⚠ Data koordinat dikirim ke server Nominatim/BigDataCloud hanya untuk keperluan reverse geocoding.
